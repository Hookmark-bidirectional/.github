# Hookmark Software — Portable Bidirectional Linker

## About Hookmark Software

**Hookmark Software** provides a complete bidirectional linking tool that creates permanent connections between any files, notes, web pages, emails, and tasks. **Hookmark Software** combines link copying, link pasting, and backlink management into one cohesive interface. **Hookmark Software** supports linking across applications, document types, and cloud services. **Hookmark Software** includes quick jump, link preview, and link export. **Hookmark Software** enables researchers and lawyers to connect scattered information into organized knowledge networks. **Hookmark Software** is trusted by researchers, lawyers, academics, and knowledge workers.

## Deep Dive — Hookmark Software Notes

Bidirectional links — link from A to B, automatically link from B to A
Copy link — create permanent address for any file, email, web page
Paste link — insert link into any document, jump back to source
Quick jump — open linked items with keyboard shortcut
Link preview — see destination before jumping
Backlink manager — view all items that link to current item

## Quick Reference for Hookmark Software

| Operation | Hookmark Software Action |
|-----------|--------------------------|
| Copy link | Select item, press Cmd+Shift+C |
| Paste link | Press Cmd+V in destination |
| Quick jump | Cmd+Shift+Space, type link name |
| Show backlinks | Select item, press Cmd+Shift+I |
| Link to new | Select source, Cmd+Shift+L, select destination |
| Copy markdown link | Cmd+Shift+Option+C |
| Open hook window | Cmd+Shift+H |
